# Discord Bot Docs

## Inviting the Bot to your Server

Add the OmniDub Discord Bot to your server by simply [clicking here](https://discord.com/api/oauth2/authorize?client_id=1176584649569476618&permissions=964757425216&scope=bot%20applications.commands).

## Omni Discord Bot: User Guide

With OmniDub, you can seamlessly translate your audio files into any language you desire. Follow the straightforward steps below to start dubbing your audio files on Discord.

### Step - 1
Use the `/dub` Slash Command in the text channel where you want to dub your audio file.

### Step - 2
Upload an Audio File
- Click on the file upload icon and choose the audio file you want to dub.

### Step - 3
Choose a Language
- After uploading the audio file, specify the language you want for the resultant audio by providing the language code or name.

### Step - 4
Press Enter to Run the Command
- Once you've uploaded the file and selected the language, press Enter to execute the `/dub` command.

### Step - 5
Bot Status: "Thinking..."
- The bot will display "Thinking..." as it processes and converts your audio file.

### Step - 6
File Display
- After a few seconds, the bot will showcase the dubbing result, providing you with the translated audio file.

> For a list of supported languages, click [here](./supported-languages).

> Note: OmniDub will notify you when the translation is ready for playback.

### Troubleshooting

The Discord bot may respond with an error message, or become infinitely stuck. If this happens, please try the `/dub` command again after a while. If the issue persists, contact our support on [Discord](https://discord.gg/8a3A2EcrEC).
